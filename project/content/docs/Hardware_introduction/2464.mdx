一.產品說明

- 1.1液晶可顯示輸入/輸出電壓，輸出電流/輸出功率/輸出容量/輸出時間；
- 1.2.數控調節，精准快捷，可升壓可降壓，輸出電壓0.5-30V任意調節，限制電流0-4A任意調節；
- 1.3.輸入端防反接保護，反接不會燒毀；
- 1.4.輸出端防倒灌，給電池充電時不用另外加防倒灌二極體；
- 1.5.可設置模組預設開啟/關閉；
- 1.6.擁有多種軟體保護機制，並且保護閾值可調。

模組工作參數超過保護閾值後，自動關閉輸出。

- 1.7.輸出紋波小，有T型濾波；
- 1.8.加厚散熱片。

三. 使用方法

3.1. 切換顯示參數—在正常介面下，短按SW，
切換顯示幕下行的顯示，顯示內容在電流A 功率
W容量Ah 時間hえ間切換・長按SW按鍵，切換
顯示幕上行的顯示，顯示內容在輸入電壓IN 輸出電壓OUT 之間切換。

3.2. 設置輸出電壓值——在正常介面下短按U/I按鍵，進入設置電壓恒流介面。可以看見設置輸出電壓值的某一位元數在閃爍，左右轉動旋轉編碼器，可調大調小。短按旋轉編碼器可以選擇設置輸出電壓的哪一位元。設置完成後，短按2下U/I按鍵，返回正常介面。或者停止操作10s後，自動回到正常介面。

3.3. 設置恒流值（即模組允許輸出的最大電流值）—在正常介面下短按U/I按鍵，進入設置電壓恒流介面。再短按U/I按鍵，切換到設置恒流值，可以看見設置恒流值的某一位在閃爍，左右轉動旋轉編碼器，可調大調小。短按旋轉編碼器可以選擇設置恒流值的哪一位。設置完成後，短按U/I按鍵，退出設置電壓恒流介面，返回正常介面。或者停止操作10s後，自動回到正常介面。

3.4. 設置模組上電預設開啟/關閉狀態——在正常介面下長按U/I按鍵進入參數設置介面。可以看到顯示 OPEN OFF或者 OPEN ON.OPEN OFF代表上

電默認關閉輸出，OPEN ON代表上電默認開啟輸出。長按旋轉編碼器可以切換兩種狀態。設置完成後，長按U/I按鍵，返回正常介面。

3.5. 設置保護參數開啟狀態和閾值——在正常介面下長按U/I按鍵進入參數設置介面。短按SW按鍵，直到出現自己想要設置的保護參數。LUP—欠壓保護閾值；OUP—過壓保護閾值；OCP—過流保護閾值；OPP——過功率保護閾值；OAP

超容量保護閾值；OHP超時保護閾值；OTP

過溫保護閾值。短按旋轉編碼器可以選擇想要設置保護參數的哪一位。長按旋轉編碼器可以設置保護參數開啟還是關閉（只有超時保護和超容量保護可以設置開啟/關閉，其他保護參數默認開啟。）。左右旋轉編碼器可以讓參數變大變小。設置完成後，長按U/I按鍵，返回正常介面。

3.6. 校準電壓電流—在正常介面下長按U/I按鍵進入參數設置介面。短按SW按鍵，直到出現帶有CAL的參數介面。帶有CAL+IN+V符號的是校準輸入電壓介面；帶有CAL+OUT+V符號的是校準輸出電壓介面；帶有CAL+OUT+A符號的是校準輸出電流介面。左右旋轉編碼器可以調節參數大小。調節完成，長按旋轉編碼器確認調節完成，此時參數值不在閃爍。長按U/I按鍵，返回正常介面。

注意：為了確保校準的準確度，校準電壓——12V以上電壓才能開始校準；校準電流——1A以上電流才能開始校準。

四.注意事項

- 4.1.模組輸入IN-禁止與輸出 OUT-短接，否則恒流功能失效。
- 4.2.請確保供電電源的功率時刻大於輸出負載所需功率！
- 4.3.模組若想滿載輸出，輸入電壓要8V以上，當輸入電壓為5V時，輸出功率約為15W。模組電流值最大4A，前提受限於最大輸出功率，比如輸出17V，電流應不大於2A。
- 4.4.本模組在超過3A、35W使用時，請加強散熱！！！
- 4.5.模組有輸入欠壓保護功能，默認是4.8V左右（可以設置），低於這個值後，會自動斷開輸出（注意是模組埠處的電壓低於欠壓保護閾值，當輸入電流比較大時，不要忽略輸入導線上的分壓）。
