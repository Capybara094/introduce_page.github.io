# Unitree Go2 Pro介紹

宇樹科技 (Unitree Robotics) 的 Go2 是一款具備高智能、敏捷運動能力的仿生消費級四足機械狗

價格: 人民幣**9997元起**

機型: Go 2 pro

配備: Go2本體、標準版電池、標準版充電器、備用足端

#### 架構圖:

![go2.png](/go2.png)

#### **標準超廣角 4D 雷射雷達使識別系統升級 200%**

Go2搭載宇樹自主研發的4D雷射雷達L1，擁有360°x90°半球形超廣角識別，超小盲區。

最小檢測距離低至0.05米，使Go2實現全地形識別。

#### **進階AI模式**

透過大規模AI模擬訓練，Go2機器人學會了倒立行走、自適應翻滾、越過障礙物等高階步態，同時具備出色的靈活性和穩定性。

#### **各種動作和姿勢**

Go2 擁有多種姿勢，例如跳躍、伸展、握手、歡呼、撲騰和坐下。

#### **電池容量及續航力提升150%**

Go2配備的電池容量提升至8000mAh，電壓提升至28.8V，提升馬達效率、動力和穩定性。

#### **ISS2.0智慧側向跟隨系統**

採用全新無線向量定位與控制技術，定位精度技術提升50%。遙控距離超過30m，結合優化的避障策略，可以讓機器人更能穿越複雜地形。

#### **3D LiDAR 測繪**

採用全新無線向量定位與控制技術，定位精度技術提升50%。遙控距離超過30m，結合優化的避障策略，可以讓機器人更能穿越複雜地形。

## **攝影機規格**

- **雙碼流支援：** 頭部相機支援 720p/15fps 與 1080p/15fps **雙碼流**（Dual-stream）技術。
- **拍攝能力：** 攝影機具備 1080p (HD) 15fps 或 720p 15fps 的**高畫質拍攝能力**。
- **光學參數：** * **光圈：** F2.2 通光孔徑。
    - **視場角 (FOV)：** 120° 超廣角。
- **影像傳輸：** 支援 App 高畫質圖傳；在無干擾且無遮擋的理想環境下，可確保 **720p 15fps** 的流暢影像傳輸品質。

**雙碼流 (Dual-stream)：** 指攝影機可以同時輸出兩種不同解析度/頻寬的串流，通常一組用於本機高畫質錄影，另一組用於 App 端的即時遠端監看。

**圖傳 (Image/Video Transmission)：** 機器人領域專有名詞，指將拍攝到的影像即時回傳至行動裝置或操作終端。
