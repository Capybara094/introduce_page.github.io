# Jetson Orin Nano™ Super 8GB

![nano.png](/nano.png)

### **【產品簡介】**

NVIDIA Jetson Orin Nano™ 超級開發套件是一款尺寸超小但功能強大的電腦，為小型邊緣裝置的生成式人工智慧樹立了全新標竿，可提供高達 67 TOPS 的人工智慧效能，比前一代提升 1.7 倍，可順暢執行各種最熱門的生成式人工智慧模型（例如：視覺轉換模型、大型語言模型、視覺語言模型等）。價格僅 249 美元，能為開發人員、學生與人工智慧自造者提供最經濟實惠且易於使用的平台，而且還有 NVIDIA 人工智慧軟體及廣大的人工智慧軟體生態系統的支援。

### **【產品特點】**

● 生成式 AI 模型效能提高 1.7 倍。

● 67 個 Sparse TOP，較之前的 40 個 Sparse TOP 大幅增加

● 102 GB/s 記憶體頻寬，較之前的 65 GB/s 記憶體頻寬有了顯著飛躍。

● CPU 速度從 1.5 GHz 提高到 1.7 GHz。

### **主要優勢**

• **卓越的 AI 效能：** 搭載 NVIDIA Ampere 架構 GPU，提供高達 40 TOPS 的運算能力。
• **開發者友善：** 支援完整的 NVIDIA AI 軟體堆疊，包括 NVIDIA JetPack™、DeepStream、Isaac 和 TAO 工具套件。
• **靈活的擴展性：** 提供多個連接埠，包括兩個相機連接埠、M.2 接口（用於 SSD 和無線網路）、以及 40 針擴充排針。

### **技術規格**

| **特性** | **規格細節** |
| --- | --- |
| **AI 效能** | 40 TOPS (INT8) |
| **GPU** | NVIDIA Ampere 架構，搭載 1024 個 CUDA 核心及 32 個 Tensor 核心 |
| **GPU 最大頻率** | 625 MHz |
| **CPU** | 6 核 Arm® Cortex®-A78AE v8.2 64 位元 CPU (1.5MB L2 + 4MB L3 快取) |
| **CPU 最大頻率** | 1.5 GHz |
| **記憶體** | 8GB 128-bit LPDDR5 (68 GB/s 頻寬) |
| **儲存空間** | 支援外部 MicroSD 記憶卡；可透過 M.2 Key M 介面擴充 NVMe SSD |
| **影片編碼** | 1080p60 (H.264) |
| **影片解碼** | 1x 4K60 (H.265) / 2x 4K30 (H.265) / 5x 1080p60 (H.265) / 11x 1080p30 (H.265) |
| **電源消耗** | 7W 至 15W (可調式) |

### 載板 (Carrier Board) 接口規格

| **接口類型** | **說明** |
| --- | --- |
| **相機 (CSI)** | 2x MIPI CSI-2 22-pin 連接埠 |
| **USB** | 1x USB 3.2 Gen 2 (Type-C)；4x USB 3.2 Gen 2 (Type-A) |
| **顯示輸出** | 1x DisplayPort 1.2 |
| **網路** | 1x 10/100/1000 Base-T 乙太網路 |
| **擴充插槽** | 1x M.2 Key M (PCIe Gen3 x4)；1x M.2 Key E (PCIe Gen3 x1)；1x MicroSD 插槽 |

| **尺寸** | 100 mm x 79 mm x 21 mm (含風扇與載板) |

### **軟體支援與生態系統**

Jetson Orin Nano 開發者套件執行 **NVIDIA JetPack™ SDK**，這是一個全方位的軟體解決方案。
• **NVIDIA JetPack:** 包含 Linux 核心 (L4T)、CUDA®、cuDNN 和 TensorRT™ 庫。
• **開發工具：** 支持完整的 AI 框架（如 PyTorch, TensorFlow）以及專為機器人設計的 **NVIDIA Isaac™** 平台。
• **簡化流程：** 配合 NVIDIA TAO（預先訓練模型微調）和 Fleet Command（邊緣管理），可顯著縮短從開發到部署的時間。
