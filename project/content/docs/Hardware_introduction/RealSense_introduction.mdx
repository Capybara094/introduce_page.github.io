# Intel® RealSense™ 景深攝影機 D435

## **【產品簡介】**

RealSense™ 深度攝影機 D400 系列採用立體視覺來計算深度。D435 是一款 USB 供電的深度相機，由一對深度感測器、RGB 感測器和紅外投影儀組成。

### **功能強大且全面的深度攝影機**

RealSense™ 深度攝影機 D435 是一款立體視覺解決方案，可為各種應用提供高品質深度偵測。

它的寬視角非常適合機器人或擴增實境和虛擬實境等應用，在這些應用中，盡可能的擴大視角至關重要。

這款外形小巧的攝影機拍攝範圍高達10公尺，可輕鬆應用到任何解決方案中，而且配置齊全，採用Intel® RealSense™ SDK 2.0，並提供跨平臺支援。

### **非常適合弱光和寬視角**

D435 結合了寬視角和全域快門感測器，是機器人導航和物體識別等應用領域的首選解決方案。

全域快門感測器提供極低的光敏感度，使機器人能夠在關燈的情況下在空間中導航。

### **簡單集成的完整套件**

深度攝影機 D435 屬於Intel® RealSense™ D400 系列攝影機，該產品系列採用了英特爾最新的深度感知硬體和軟體，並將其封裝到易於集成的產品中。

Intel® RealSense™ D400 系列攝影機非常適合希望將深度感知引入設備的開發人員、製造商和創新人士，它提供了簡單的現成集成，並支援全新一代智慧視覺安裝設備。

## **【產品特色】**

- 強大的視覺處理器，採用 28 奈米 (nm) 工藝技術，支援多達 5 個 MIPI 相機串行接口雙通道，以計算即時深度影像並加速輸出。
- 新的高級立體深度演算法可實現準確的深度感知。
- 可以捕獲分辨率高達 1280 x 720 的影像之間的差異。
- 支援跨平台、開源的Intel® RealSense™ SDK 2.0。
- 用於影像調整和縮放顏色數據的專用彩色影像訊號處理器。
- 主動紅外線投影儀照亮物體以增強深度數據。

    - 深度輸出分辨率：高達1280×720
    - 深度輸出幀率：高達 90 fps
