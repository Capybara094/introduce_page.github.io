XL4015 DC-DC可調降壓模組（帶數顯電壓表）

【尺寸】66mm（長）*39mm（寬）*18mm（高）

【輸入電壓】DC 4v~38v（輸入電壓需比輸出電壓高1.5V以上）

【輸出電壓】DC 1.25v~36v（可調）

【輸出電流】5A（最大），長時間使用建議4.5A以內（附贈散熱片）

【輸出功率】75W（最大）

【轉換效率】96%（最高）

【開關頻率】180KHz

【工作溫度】-45~+85°C

【保護類型】短路保護、過熱保護

【其他說明】

1.銅柱*4、附贈散熱片*1（建議安裝）

【操作說明】

1. 大體積5*5輕觸按鍵切換測量輸入或輸出電壓，並有指示燈顯示正在測量的是哪端電壓。初次開機默認顯示輸入電壓，並且輸入指示燈IN亮；輕觸按鈕，則顯示輸出電壓，並且輸出指示燈OUT亮。而且保持設定，即使是斷電再開機。
2. 若模組輸出電壓一直等於輸入電壓，請先將電位器逆時針旋轉20圈以上即可正常工作！
3. 逆時針降壓，順時針升壓
