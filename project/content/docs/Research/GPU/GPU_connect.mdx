title: GPU連線教學
---

1. 在你自己的電腦安裝 **VS Code**。
2. 安裝外掛 **Remote - SSH**。
3. 設定連線 (點左下角綠色圖示 -> Connect to Host -> 輸入 user@ip -p port)。

解決方法：改用「設定檔 (Config File)」連線 (最穩定的做法)

不要直接在對話框輸入那一長串指令，我們直接把設定寫進檔案裡，這樣以後只要點一下就能連線，不用重打。

教學步驟如下：

### 第一步：開啟設定檔

1. 在 VS Code 畫面，按鍵盤上的 **F1** (或 Ctrl + Shift + P)。
2. 輸入 **ssh config**。
3. 選擇 **Remote-SSH: Open SSH Configuration File**。
4. 接著會跳出幾個路徑讓你選，通常選**第一個** (類似 C:\Users\你的名字\.ssh\config)。

### 第二步：修改設定內容

打開檔案後，你可能會看到裡面是空的，或者有一些亂掉的設定。**請在檔案的最下面**，複製貼上以下內容（請根據你的真實資訊修改）：

```c
Host _________
    HostName ________
    User _______

```

**⚠️ 注意事項 (請檢查)：**

1. **Host**: 這只是個暱稱，你可以隨便取 
2. **HostName**: 這裡**只能**填 IP ，**不要**在後面加 -p。
3. **User**: 填你的帳號 。

### 第三步：存檔並重新連線

1. 按 Ctrl + S 存檔，然後關閉這個檔案。
2. 點擊 VS Code 左側邊欄那個「遠端螢幕」圖示。
3. 你現在應該會看到一個新的”(HOST NAME)”出現在清單裡。
4. 滑鼠移過去，點擊右邊出現的 **「小視窗圖示 (Connect in New Window)」** 或直接點它連線。

---

### 第六步：長時間訓練不中斷 (Tmux)

如果你的訓練要跑好幾個小時，電腦休眠網路斷掉，程式就會死掉。請使用 tmux。
