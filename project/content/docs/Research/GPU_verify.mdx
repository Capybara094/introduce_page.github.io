### 3.在伺服器建立新檔案

現在左側邊欄顯示的就是 **學校伺服器的硬碟** 了。

1. 在左側空白處按右鍵 -> **New File (新增檔案)**。
2. 檔名輸入 檔名(ex. test.py)。
3. 貼上以下程式碼

```c
import torch
import os

# 1. 檢查 PyTorch 版本
print(f"PyTorch version: {torch.__version__}")

# 2. 檢查 CUDA (顯卡支援) 是否可用
print(f"CUDA available: {torch.cuda.is_available()}")

# 3. 顯示目前程式能看到的顯卡數量
print(f"Device count: {torch.cuda.device_count()}")

# 4. 顯示目前抓到的顯卡名稱 (如果有抓到的話)
if torch.cuda.is_available():
    # 這裡的 0 是指程式看到的「第1張」，不是物理上的 ID
    print(f"Current device: {torch.cuda.get_device_name(0)}")
else:
    print("No GPU detected for this process.")
```

###4.按 Ctrl + S 存檔**。
